# SimpleTodoApp

A lightweight, browser‑based Todo application built with plain HTML, CSS, and JavaScript. It lets users add, complete, delete, and clear tasks, with data persisted in the browser's LocalStorage and a responsive design that works on both desktop and mobile devices.

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## Features

- Add new tasks
- Mark tasks as completed
- Delete individual tasks
- Clear all completed tasks
- Persist tasks using LocalStorage
- Responsive design for mobile and desktop

## Setup & Running

1. Clone the repository.
2. Open `index.html` in a browser (no server needed).

## Usage

- **Add a task**: Type into the input field and press **Enter** or click the add button.
- **Complete a task**: Click the checkbox next to the task.
- **Delete a task**: Click the delete (✖) button on the task.
- **Clear completed tasks**: Click the **Clear Completed** button.

## LocalStorage Persistence

- The app stores tasks in the browser's **LocalStorage** (key: `todos`). This data persists across page reloads and browser restarts.
- To reset the list, clear the browser's LocalStorage for this site (e.g., via DevTools → Application → Local Storage → **Clear**).

## Responsive Design

- The layout adapts to various screen sizes, providing a mobile‑friendly view with stacked elements and touch‑friendly controls.
